klay mu's quick guide to self-hosting

This website is hosted on a Raspberry Pi, but if those aren't available where and when you are, a good alternative is the Rock 64. I also recommend some sort of external hard drive so you can hoard all the data you want.

First, go ahead and install whatever flavor of linux you're going to use, I recommend armbian. If your setup is wireless, install 'avahi-daemon' to make it easier to connect over LAN.
